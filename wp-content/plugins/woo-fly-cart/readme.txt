== Description ==

WooCommerce Fly Cart is a WooCommerce interaction mini cart with many styles and effects. You can change the product quantity or remove product without reload the page.

= Features =

- Auto open when click Add to cart or manual the button
- AJAX change quantity or remove product
- 5 pre-defined positions & effects
- 5 pre-design styles
- Unlimited color
- Fully responsive

= Need support? =

Visit [plugin documentation website](https://wpclever.net "plugin documentation")

== Installation ==

1. Please make sure that you installed WooCommerce (https://wordpress.org/plugins/woocommerce/)
2. Go to Plugins in your dashboard and select "Add New"
3. Search for "WooCommerce Fly Cart", Install & Activate it
4. Go to WooCommerce / Fly Cart and customize as you want
5. Now, whenever you add product into the cart, the mini cart will be show
