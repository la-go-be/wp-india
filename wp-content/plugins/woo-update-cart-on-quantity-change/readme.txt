Woo update cart on quantity change plugin is compatible with latest version of woocommerce. Built with the security standards for the most recent WP version.

== Description ==
Woo Update Cart On Quantity Change plugin allows wooCommerce cart to get auto updated when user changes quantity of any product in cart. No more hassel to click on update cart button manually.

= Note =
This plugin is ready to use with any theme.

== Installation ==

 Quick Setup


 1) Install plugin through backend of your Wordpress site or via FTP.
 

 Having trouble with installation? Just check our website with free videos on installation and setup.

 From Wordpress Backend (automatic):
 Go to `Plugins -> Add new` -> click on Upload Plugin and upload the File address-autocomplete-using-google-place-api.zip,
 or by Search Plugins, type in:`woo-update-cart-on-quantity-change` and click on ìnstall`

 Manual Setup:
 1. Upload `woo-update-cart-on-quantity-change` Folder from unzip address-autocomplete-using-google-place-api.zip to the `/wp-content/plugins/` directory
 2. Activate the plugin through the 'Plugins' menu in WordPress
